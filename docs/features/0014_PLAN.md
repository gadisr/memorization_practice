# Technical Plan: Google SEO and Analytics Integration

## Brief Context

The user requested to add Google SEO and analytics to the BLD Memory Trainer website. This involves implementing Google Analytics 4 (GA4) tracking, Google Search Console integration, and comprehensive SEO optimization including meta tags, structured data, sitemap generation, and performance optimization for better search engine visibility and user analytics.

## Current State Analysis

### Existing Infrastructure:
- ✅ **Frontend**: TypeScript/HTML5/CSS3 static website
- ✅ **Main HTML files**: `public/index.html`, `public/intro.html`, `public/onboarding.html`
- ✅ **Deployment**: Static files served via HTTP server (Python serve.py or npm serve)
- ✅ **Domain**: blindfoldcubing.com (based on HTTPS deployment docs)
- ✅ **Authentication**: Firebase with Google OAuth
- ✅ **Content**: Educational content about blindfold cubing with structured pages

### Technology Stack:
- Frontend: TypeScript, HTML5, CSS3
- Build: TypeScript compiler, Vite
- Deployment: Static file hosting
- **NEW**: Google Analytics 4, Google Search Console, SEO optimization

## Implementation Details

### SEO Meta Tags Structure:
```html
<!-- Basic SEO -->
<meta name="description" content="BLD Memory Trainer - Improve your blindfold cubing visualization skills with structured training drills">
<meta name="keywords" content="blindfold cubing, BLD, memory training, Rubik's cube, visualization, speedcubing">
<meta name="author" content="BLD Memory Trainer">

<!-- Open Graph -->
<meta property="og:title" content="BLD Memory Trainer - Blindfold Cubing Memory Training">
<meta property="og:description" content="Structured training drills to improve your blindfold cubing visualization skills">
<meta property="og:type" content="website">
<meta property="og:url" content="https://blindfoldcubing.com">
<meta property="og:image" content="https://blindfoldcubing.com/og-image.jpg">

<!-- Twitter Cards -->
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="BLD Memory Trainer">
<meta name="twitter:description" content="Improve your blindfold cubing skills with structured training">
```

### Google Analytics 4 Integration:
```html
<!-- Google Analytics 4 -->
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
```

## Environment Configuration

### Required Environment Variables:
- `GA_MEASUREMENT_ID` - Google Analytics 4 measurement ID
- `GOOGLE_SITE_VERIFICATION` - Google Search Console verification code
- `SITE_URL` - Base URL for canonical links and structured data

### Google Services Setup:
1. **Google Analytics 4**: Create GA4 property and obtain measurement ID
2. **Google Search Console**: Verify domain ownership and submit sitemap
3. **Google Tag Manager** (optional): For advanced tracking management
