# SEO Content Strategy

## Overview
This document outlines the content strategy based on keyword research and competitor analysis.

## Application Context

### Website: blindfoldcubing.com
- **Primary Function**: Free web-based training application for blindfold cubers
- **Key Features**: 
  - Flash Pairs (individual letter pair visualization)
  - Edge Notation Drill (recognize Speffz letters from edge pieces)
  - Corner Notation Drill (recognize Speffz letters from corner pieces)
  - Edge Tracing Drill (trace edge cycles from scrambles)
  - Edge Memorization (memorize edge colors, recall as letters)
  - Corner Memorization (memorize corner colors, recall as letters)
  - Progress tracking and analytics dashboard
  - Session management with recall accuracy measurement
- **Current Pages**:
  - Main App (/): Training application interface
  - Intro Guide (/intro.html): Beginner's guide to blindfold cubing
  - Onboarding (/onboarding.html): Interactive tutorial
- **Technology**: Offline-first web app, mobile-responsive, browser-based
- **Target Audience**: Beginners to advanced blindfold cubers

### Current Content Overview
- **Main App Page**: Training interface with drill selection, session management, dashboard
- **Intro Guide**: Comprehensive beginner's guide covering BLD basics, memory systems, learning progression, Speffz notation
- **Onboarding**: Interactive tutorial for new users
