# SEO Keyword Research

## Overview
This document contains the ChatGPT prompts and research strategy for keyword research for blindfoldcubing.com.

## Application Context

### Website Information
- **Domain**: blindfoldcubing.com
- **Application Name**: BLD Memory Trainer / Blindfold Cubing Training
- **Primary Purpose**: Web-based training application for blindfold cubers to improve visualization speed, story fluency, and recall accuracy
- **Target Audience**: 
  - Beginners learning blindfold cubing (BLD)
  - Intermediate cubers improving their BLD skills
  - Advanced cubers seeking structured practice tools
  - Speedcubers focused on memory training and visualization

### Core Features
- **Training Modes**: 
  - Flash Pairs (single letter pairs, 12 pairs, Vividness metric)
  - Edge Notation Drill (recognize Speffz notation from edge pieces, 24 pairs)
  - Corner Notation Drill (recognize Speffz notation from corner pieces, 24 pairs)
  - Edge Tracing Drill (trace edge cycles from scramble strings)
  - Edge Memorization (memorize edge colors and recall as letters, 12 pairs)
  - Corner Memorization (memorize corner colors and recall as letters, 12 pairs)
- **Quality Metrics**: Vividness (1-5) for visualization clarity, Flow (1-3) for story continuity
- **Features**: Automatic timing, session tracking, recall accuracy measurement, progress dashboard, CSV export, keyboard shortcuts, offline-first, mobile-responsive
- **Technology**: Web application (HTML/TypeScript), works offline, uses browser localStorage, Firebase authentication

### Current Content Structure
- **Main App Page** (/): Training application with drill selection and session management
- **Intro/Guide Page** (/intro.html): Comprehensive beginner's guide to blindfold cubing, covers:
  - What is blindfolded cubing
  - The process (inspection, blindfold, execution)
  - Memory system and letter pairs
  - Learning progression (5 steps)
  - Memory Palace technique
  - Speffz notation system
  - Cognitive benefits
- **Onboarding Page** (/onboarding.html): Interactive tutorial for new users

### Key Terminology
- BLD (Blindfolded)
- Letter pairs
- Speffz notation
- Memory Palace technique
- Visualization training
- Story building
- Recall accuracy
- Edge tracing
- Corner tracing
- Edge cycles
- Color memorization

### User Goals and Pain Points
- **Goals**: Improve BLD memorization speed, increase recall accuracy, develop visualization skills, practice letter pair systems, track progress over time
- **Pain Points**: Lack of structured practice tools, difficulty tracking improvement, need for progressive training, want to practice offline, need feedback on visualization quality

## Phase 1: Seed Keywords

### ChatGPT Prompt 1.1
```
I'm doing SEO keyword research for blindfoldcubing.com, a web-based training application for blindfold cubers.

Website Context:
- Domain: blindfoldcubing.com
- Application: BLD Memory Trainer - helps cubers improve visualization speed, story fluency, and recall accuracy
- Target Audience: Beginners to advanced blindfold cubers learning or improving BLD skills
- Core Features: Multiple training drills (Flash Pairs, Edge/Corner Notation, Edge Tracing, Memorization drills), progress tracking, session management, CSV export, offline-first web app
- Current Content: Main training app, comprehensive beginner's guide (/intro.html), interactive onboarding tutorial
- Key Topics: Blindfold cubing (BLD), letter pairs, Speffz notation, memory palace technique, visualization training, edge/corner tracing, color memorization

Give me 30 semantically relevant but unique topics/keywords under the main category of 'blindfold cubing' or 'BLD cubing' that would be relevant for users searching for:
1. Training tools and practice applications
2. Learning how to solve Rubik's cube blindfolded
3. Improving BLD memory and visualization skills
4. Letter pair systems and memorization techniques
5. Speffz notation and edge/corner tracing

Give me the result in a table with 5 columns: Keyword, Search Intent (Informational/Navigational/Transactional), Difficulty (Low/Medium/High), Relevance to Training App (1-10), Priority for Immediate Focus (High/Medium/Low).
```

### Follow-up Prompt 1.1
```
From the list of 30 seed keywords you provided, identify the top 5 you think are most important for immediate focus and explain why. 

Consider:
- Search volume potential (are people actively searching for this?)
- Competition level (can we rank for this?)
- Relevance to our training application (does it match our features?)
- User intent alignment (are searchers looking for what we offer?)
- Commercial/tool-seeking intent (are they looking for practice tools?)

For each of the top 5, suggest:
1. Which page on blindfoldcubing.com should target this keyword
2. What content adjustments would help us rank better
3. What features of our training app are most relevant to this keyword
```
