# Logo Design Prompt for BLD Training Platform

## Application Overview
BLD Training Platform for blindfold Rubik's cube solving at **blindfoldcubing.com**. Provides memory training, cube tracing drills, notation training, and progress tracking for comprehensive BLD skill development.

## Core Visual Requirements

### Primary Element: 3x3 Rubik's Cube
- **Central focal point** with standard colors (white, yellow, blue, green, orange, red)
- **Scrambled state** (not solved) in **3D perspective** showing depth
- **Partially covered with blindfold** material (elegant draping, not comedic)
- **Wireframe arrows** showing U (Up), R (Right), B (Back) movements in clean, technical style
- Dynamic positioning suggesting mid-solve motion

### Supporting Elements
- **Training indicators**: Letter notation elements, tracing lines, memory connections
- **Athletic aesthetic**: Professional sports training vibe with clean, modern design
- **Technical precision**: Grid patterns, measurement indicators

## Design Guidelines

### Colors
- **Primary**: Deep blues/purples (focus, concentration)
- **Accent**: Bright orange/yellow/green (energy, success)
- **Neutral**: Clean whites, light grays
- **Cube colors** as accent elements

### Typography
- Modern, clean sans-serif
- Athletic styling, readable at small sizes
- Must work with "blindfoldcubing.com" branding

### Style
- Professional yet approachable
- Technical precision meets athletic training
- Scalable from favicon (16x16) to large displays
- Works in color and monochrome

## Technical Requirements
- **Formats**: SVG (vector), high-res PNG
- **Sizes**: Favicon to 512x512 app icon
- **Versatility**: Browser tabs, social media, search results
- **Monochrome compatibility** required

## Brand Personality
Convey: Intelligence, technical precision, athleticism, focus, comprehensive training, progress

## Avoid
- Overly complex designs that don't scale
- Cartoonish blindfold imagery
- Generic cube designs
- Cluttered compositions

## Deliverable
Logo that immediately communicates comprehensive BLD training platform for mental athletes, combining 3D scrambled cube, elegant blindfold integration, wireframe movement indicators, and technical training aesthetics suitable for blindfoldcubing.com branding.
