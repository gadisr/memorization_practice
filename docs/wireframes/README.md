## Key Recommendations

### Top Priority CTAs

1. **Progress Teaser** (Show after 1-2 drills)
   - Display actual drill results
   - Create investment through activity
   - Prompt to save progress

2. **PDF Download Gate** (Always visible)
   - High-value content
   - Clear sign-in requirement
   - Educational = high intent

### Homepage Layout Order

1. Hero section (value prop + start drill)
2. Quick drill interface (let them try)
3. Progress teaser CTA (after activity)
4. Drill modules grid
5. PDF tutorial section (sign-in gate)
6. Social proof
7. Features overview

### Conversion Principles

- ✅ Value first, ask second
- ✅ Multiple conversion points
- ✅ Low friction (Google OAuth)
- ✅ Show actual results/progress
- ✅ Create urgency with loss aversion
- ✅ Never gate initial experience
